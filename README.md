# 💕 Love App

Романтическое веб-приложение для пар, созданное на Vue 3. Считает дни вместе, хранит общие планы в облаке, показывает фотографии и предлагает пройти тест о второй половинке.

## 📋 Описание проекта

Love App — это интерактивное приложение, которое помогает сохранять тёплые моменты отношений. Здесь можно отслеживать количество дней вместе, планировать совместные дела, делиться фотографиями и проверять, насколько хорошо вы знаете друг друга.

## ✨ Функциональность

### 💑 Главный экран
- Приветствие с именем партнёра
- Кнопки для доступа ко всем функциям
- Падающие сердечки на фоне
- Случайные романтические сообщения

### 📅 Счётчик дней (DayCounter)
- Количество дней, недель и месяцев вместе
- Прогресс-бар до годовщины
- Дата начала отношений и текущая дата
- Вдохновляющие сообщения в зависимости от срока

### 💬 Случайное сообщение (LoveMessage)
- Показывает случайное романтическое сообщение
- Исчезает через 3 секунды
- Более 15 вариантов признаний

### 🖼️ Галерея фотографий (PhotoGallery)
- Горизонтальная прокрутка фотографий
- Открытие фото в полноэкранном режиме
- Навигация по фото (стрелки)
- Подписи к каждому фото

### ❤️ Тест о партнёре (LoveTest)
- 5 вопросов о второй половинке
- Проверка ответов с подсветкой (зелёный/красный)
- Подсчёт правильных ответов
- Подарок после прохождения (анимированное сообщение)

### 📆 Планировщик с облаком (PlanCalendar)
- Календарь с возможностью добавлять планы на любой день
- Цветовая маркировка планов (6 цветов)
- Визуальные индикаторы на днях с планами
- Редактирование и удаление планов
- **Облачная синхронизация через Firebase** — данные доступны на любом устройстве

## 🛠️ Технологии

- **Vue 3** (Composition API, `<script setup>`)
- **Firebase Realtime Database** — облачное хранение планов
- **Vite** — сборка проекта
- **Font Awesome** — иконки
- **CSS3** — анимации, градиенты, адаптивный дизайн
